A client-side React app for building weekly class and event schedules. Add events with times, day recurrence, and custom colors. Events appear in a seven-column grid with overlapping events sitting side-by-side. Export the schedule as PNG or PDF (letter, legal, A4). Import and export full schedules as JSON. Comes with 15 theme families in light and dark modes.
